# NSD Backend

## Overview
This directory contains the backend services for NSD
(Network Suspicious Detection system).

The backend is responsible for:
- Device data ingestion
- Event processing
- Detection engine integration
- Alert management
- Case management
- User authentication
- API services for frontend dashboard

## Main Components
- REST API
- Detection Engine Interface
- Alert System
- Database Access Layer
- Authentication / Authorization
- Audit Logging

## Planned Technology Stack
- Backend: Python (FastAPI) or Node.js
- Database: PostgreSQL
- Message Queue: RabbitMQ or Kafka
- Cache: Redis
- Deployment: Docker / Cloud
- Authentication: JWT / OAuth / Zero Trust

## API Categories
- Authentication API
- Device API
- Event API
- Alert API
- Case Management API
- User Management API
- Admin API

## Future Features
- Real-time alerts (WebSocket)
- Machine learning risk scoring
- Video / Camera integration
- External system integration
