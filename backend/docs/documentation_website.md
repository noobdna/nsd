NSD – Documentation & Website

Overview

This document describes the documentation system and public website
for the NSD (Network Suspicious Detection) project.

The documentation site serves multiple purposes:
	•	System documentation
	•	Technical architecture explanation
	•	API documentation
	•	User manuals
	•	Project roadmap
	•	Pilot test information
	•	Public project introduction
	•	Developer onboarding

The documentation website acts as the central knowledge base for NSD.

⸻

Documentation Goals

The documentation system should:
	•	Explain the NSD system architecture
	•	Document APIs and database structure
	•	Provide deployment instructions
	•	Provide operator manuals
	•	Provide developer guides
	•	Publish project roadmap
	•	Support open source collaboration
	•	Provide public project information
	•	Serve as investor / partner explanation material
