
# NSD – Use Cases

## Overview
NSD (Network Suspicious Detection) can be used in many areas including personal safety, corporate security, smart cities, healthcare, and law enforcement support.

## Use Cases

### 1. Stalker Detection
Detect repeated proximity of a specific device or person to a target location or person.
If the same device appears repeatedly near a person’s home, workplace, or school, the system raises a risk alert.

### 2. Suspicious Person Monitoring
Monitor unusual movement patterns around homes, schools, offices, or restricted areas.
Detect loitering, repeated visits, or unusual time patterns.

### 3. Child Safety Monitoring
Alert when a child leaves a safe zone or when unknown devices repeatedly approach the child’s device.

### 4. Elderly / Dementia Monitoring
Detect wandering patterns and notify family or caregivers when unusual movement patterns are detected.

### 5. Corporate Security
Detect suspicious access patterns, repeated entry attempts, unusual network activity, or unauthorized devices.

### 6. Smart City Security
City-wide anomaly detection using cameras, sensors, mobile devices, and network data.

### 7. Domestic Violence Protection
Alert when a restricted person approaches a protected person within a defined distance.

### 8. Parole / Probation Monitoring
Monitor movement restrictions and detect violations automatically.

### 9. Campus / School Security
Detect suspicious persons repeatedly approaching school areas or specific students.

### 10. Event Security / Crowd Monitoring
Detect unusual movement patterns, crowd anomalies, or suspicious repeated presence at events.
