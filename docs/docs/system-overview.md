# NSD System Overview

## Purpose
NSD (Network Suspicious Detection) is a system designed to detect suspicious behavior,
monitor distance risks, and provide alert notifications using mobile, IoT, and cloud systems.

## Target Use Cases
- Stalker detection
- Child safety
- Elderly / dementia monitoring
- School safety
- Community safety
- Suspicious network behavior detection

## System Flow
Device → Cloud → Detection Engine → Alert → Dashboard
