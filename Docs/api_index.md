# NSD – API Index

## Overview

This document lists all APIs available in the NSD (Network Suspicious Detection) system.

The API layer is organized into functional modules that support device management, event ingestion, alert handling, investigation workflows, and system administration.

---

## API Modules

### 1. Authentication API
Handles login, logout, token refresh, and user session management.

### 2. User Management API
Handles user accounts, roles, permissions, and organization membership.

### 3. Device API
Handles device registration, updates, status tracking, and ownership.

### 4. Event API
Handles event ingestion from devices, logs, and sensors.

### 5. Risk Scoring API
Calculates and retrieves risk scores for devices, users, and events.

### 6. Alert API
Handles alert creation, updates, assignment, comments, and escalation.

### 7. Case API
Handles investigation cases, evidence, notes, and case lifecycle.

### 8. Evidence API
Handles file uploads, logs, images, and investigation evidence storage.

### 9. Report API
Generates investigation reports, incident summaries, and statistics.

### 10. Admin API
System configuration, rule management, and system settings.

### 11. Audit Log API
Tracks all system actions and changes for auditing and security.
